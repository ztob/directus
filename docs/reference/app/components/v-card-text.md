# Card Text

This component is used to give a `v-card` an already stylized text.

```html
<v-card>
	<v-card-text>This card can do nothing!</v-card-text>
</v-card>
```

## Reference

#### Slots

| Slot      | Description             | Data |
| --------- | ----------------------- | ---- |
| _default_ | Where your text will go |      |
