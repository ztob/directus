# List Item Content

Add properly formatted content to the `v-list-item`.

## Reference

#### CSS Variables

| Variable                            | Default                    |
| ----------------------------------- | -------------------------- |
| `--v-list-item-content-padding`     | `9px 0`                    |
| `--v-list-item-content-font-family` | `var(--family-sans-serif)` |

#### Slots

| Slot      | Description            | Data |
| --------- | ---------------------- | ---- |
| _default_ | Where the content goes |      |
