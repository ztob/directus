# Card Title

This component is used to give a `v-card` an already stylized title.

```html
<v-card>
	<v-card-title>My Own Title</v-card-title>
</v-card>
```

## Reference

#### Slots

| Slot      | Description             | Data |
| --------- | ----------------------- | ---- |
| _default_ | Where the title will go |      |
