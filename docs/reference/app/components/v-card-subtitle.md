# Card Subtitle

This component is used to give a `v-card` an already stylized subtitle.

```html
<v-card>
	<v-card-subtitle>A less important subtitle</v-card-subtitle>
</v-card>
```

## Reference

#### Slots

| Slot      | Description                | Data |
| --------- | -------------------------- | ---- |
| _default_ | Where the subtitle will go |      |
